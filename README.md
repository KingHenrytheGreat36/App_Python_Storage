# Python Secure(ish) Storage

A simple storage system written in Python.
Includes logig and account creation.
This project is currently in development as of July 16, 2026.

## Features
- User login system
- Account creation
- Settings menu   (COMMING SOON)  
- Shelve-based class storage backen
- Hashed passwords

## How to Run
1. Install Python 3  
1.2. Install Tkinter, if needed
2. Clone the repository  
3. Run the main script: App.py
